# Phase 2: Daily Usage

> 💡 **Goal**: Complete file operations, code reading, and text generation independently

## 🎯 Phase Objectives

- Get familiar with OpenCode's interface and core operations
- Master session management (create, switch, undo, export)

## ⏳ Estimated Time

45 minutes ~ 1 hour

## 🚀 Before You Start

- Completed Phase 1 (can have normal conversations with AI)
- Have a test project or folder ready

| [2.1b How to Copy Content](./01b-copy-paste) | Copy, selection and paste habits | 5 min |
| [2.1c AI's Basic Tools](./01c-basic-tools) | Usage and roles of 6 major tools | 20 min |

| [2.5 Environment Management](./05-env-management) | Model list, statistics, identity | 10 min |
| [2.6 Git Basics](./06-git-basics) | First commit and sync to GitHub | 15 min |

## 🎒 What You'll Gain

After completing this phase, you will:

1. Proficiently use `@` to reference files, `!` to execute commands, and `/` slash commands
2. Be able to create and manage multiple conversation sessions
