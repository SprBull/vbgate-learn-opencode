## Contributing

OpenCode is open source and we welcome contributions!

- **Report bugs**: Open an issue with reproduction steps
- **Suggest features**: Start a discussion first
- **Submit PRs**: Check existing issues, fork the repo, and submit

<a href="https://github.com/anomalyco/opencode/blob/main/CONTRIBUTING.md" class="link-button">Read Contributing Guide</a>

## Community Guidelines

<div class="rules-section">

1. **Be respectful**: Treat everyone with kindness and patience
2. **Stay on topic**: Keep discussions related to OpenCode and AI-assisted development
3. **No spam**: Avoid unsolicited promotions or off-topic content
4. **Search first**: Check existing discussions before asking questions
5. **Be constructive**: Provide helpful feedback and actionable suggestions

</div>
