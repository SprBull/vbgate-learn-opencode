## Core Concepts

### AIDA Model

| Stage | English | Description | Copy Purpose |
|-----|------|------|---------|
| A | Attention | Grab attention | Hook with headline |
| I | Interest | Build interest | Address pain points |
| D | Desire | Create desire | Highlight benefits |
| A | Action | Drive action | CTA for conversion |

### Copy Types Overview

| Type | Characteristics | Word Count |
|-----|------|------|
| Social Media Post | Short, visual-friendly | 50-150 words |
| Blog/Newsletter | Story-driven, in-depth | 1000-3000 words |
| Product Page | Structured, benefit-focused | 2000+-12000+ words |
| Tagline | Minimal, memorable | 5-20 words |

## Follow Along

### Step 1: Analyze Product Selling Points

**Why**  
Great copy starts with understanding the product.

Switch to Plan mode:

```
I'm promoting "Nike Air Zoom Pegasus" running shoes, priced at $130.
Help me analyze:
1. Target user persona (age, occupation, pain points)
2. Core selling points (3 key points)
3. Differentiation from competitors
4. Possible purchase concerns and how to address them
```

### Step 2: Generate Clickbait Headlines

**Why**  
Headlines determine 80% of click-through rates.

```
Generate 10 headlines for these shoes, requiring:
1. Use numbers (e.g., "3 Reasons")
2. Create curiosity (e.g., "Why...")
3. Lead with pain points (e.g., "Still dealing with...")
4. Results-oriented (e.g., "50% faster...")

At least 2-3 of each type
```

### Step 3: Write AIDA-Structured Copy

<AdInArticle />

**Why**  
Structured copy is more persuasive.

Switch to Build mode:

```
Write a social media post (under 150 words) using the AIDA model:

【Attention】One-line hook to grab attention
【Interest】Describe user pain points
【Desire】Show how the product solves them
【Action】Promo info + call to action

Save as copy/social-post.md
```

### Step 4: Generate A/B Variants

**Why**  
Multi-version testing finds the best performer.

```
Based on the previous copy, generate 3 variant versions:
- Version A: Emphasize "comfort" experience
- Version B: Emphasize "value for money"
- Version C: Emphasize "trusted by athletes"

Save as copy/AB-test-versions.md
```

### Step 5: Complete Copy Suite

**Why**  
Generate multi-channel assets in one go.

```
Generate a complete copy suite for these shoes:
1. Tagline (under 10 words)
2. Social media post (150 words)
3. Instagram caption (300 words + 5 hashtags)
4. Product page copy (800 words, with benefit sections)

Save all to the copy/ directory
```

## 📋 Magic Prompts

### 📢 AIDA Copy Generator
> Expected Result: Generate structured marketing copy that boosts conversion

```
## Role
You are a senior copywriter skilled in writing high-converting marketing copy using the AIDA model.

## Task
Write marketing copy for a product using the AIDA model.

## Input
### Required
- Product name: 【name】
- Core selling points: 【1-3 points】
- Target audience: 【user persona】

### Optional
- Copy context: 【social/blog/product page?】(default: social)
- Word limit: 【word count?】(default: 150 words)
- Promo info: 【limited time/quantity/discount?】

## Output Format
Output in AIDA four-stage structure:

### A - Attention
> One line to grab eyeballs (can use questions/numbers/contrast)

【Opening copy】

### I - Interest
> Describe target user's pain points to create resonance

【Pain point description】

### D - Desire
> Show how the product solves pain points + unique advantages

【Product value】

### A - Action
> Promo info + clear call to action (limited time/quantity/exclusive)

【CTA copy】

---

**Complete Copy**:
【Integrated full copy】

## Constraints
- ✅ Each stage has a clear function
- ✅ Conversational language, avoid formal tone
- ✅ CTA should be specific with urgency
- ❌ Avoid exaggerated claims or false advertising
- ❌ Avoid absolute terms like "best" or "#1"
```
