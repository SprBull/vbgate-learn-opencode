# Phase 3: Efficient Workflow

> 💡 **Goal**: Master Plan/Build modes, Agent collaboration, and project initialization

## 🎯 Phase Objectives

- Understand the difference between Plan mode and Build mode
- Learn to call different Agents to complete tasks
- Use `/init` to help AI understand your project

## ⏳ Estimated Time

45 minutes ~ 1 hour

## 🚀 Before You Start

- Completed Phase 1 and Phase 2
- Have a real project ready (code or writing project both work)

| Course | Content | Duration |

| [3.2 Meet Agents](./02-agents) | Your dedicated AI team | 15 min |
| [3.3 Project Initialization](./03-init) | Help AI understand your project | 15 min |

## 🎒 What You'll Gain

After completing this phase, you will:

1. Know when to use Plan mode and when to use Build mode
2. Be able to call different Agents for specialized tasks
3. Be able to generate project rule files with `/init`
