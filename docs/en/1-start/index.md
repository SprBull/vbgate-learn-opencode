# Phase 1: Quick Start

> 💡 **Goal**: Complete installation and have a successful conversation within 30 minutes

## 🎯 Phase Objectives

- Successfully install OpenCode without errors
- Configure network connectivity (important for users in China)
- Complete your first "Hello World" conversation

## ⏳ Estimated Time

25 ~ 45 minutes

## 🚀 Before You Start

- A computer with internet access (Windows / macOS / Linux)
- A terminal application (recommended: WezTerm / iTerm2 / Windows Terminal)
- A cup of water ☕

| Lesson | Content | Duration |
|--------|---------|----------|
| [1.1 What is OpenCode](./01-intro) | Understand what OpenCode can do | 3 min |

| ↳ [1.2b Troubleshooting](./02b-install-troubleshoot) | Installation issue resolution | 10 min |
| [1.3 Network Config](./03-network) | Essential for users in China | 10 min |
| [1.4 Connect Models](./04-connect) | Send your first message | 10 min |
| [1.5 Auto Update](./05-update) | OpenCode updates automatically by default | 5 min |

## 🎒 What You'll Gain

After completing this phase, you will:

1. Run the `opencode` command in your terminal
2. Have network configured (or choose a China-accessible model)
3. Have your first conversation with AI
