## ✨ 新功能：消息删除 API

### 这次更新了什么？

新增 `DELETE /session/:id/message/:messageId` 端点，可以通过 Remote API 删除会话中的消息。

### 这有什么用？

- **清理对话**：删除不需要的历史消息，保持对话整洁
- **重新生成**：删除某条消息后让 AI 重新回答
- **自动化脚本**：在自动化工作流中管理对话内容

### 如何使用？

```bash
# 删除指定会话中的某条消息
curl -X DELETE http://localhost:4096/session/abc123/message/msg456
```
